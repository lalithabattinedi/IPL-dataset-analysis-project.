IPL Deliveries Dataset – Beginner-Friendly Data Analysis

This project explores the IPL Deliveries dataset using Python and Pandas. It focuses on answering 13 simple cricket analytics questions related to batting, bowling, dismissals, and team performance. The notebook is written in a beginner-friendly way so anyone learning data analysis can follow along.

Project Objectives

• The analysis focuses on extracting insights such as:

• Number of unique batsmen

• Total runs and extra runs in IPL history

• Types of dismissals with counts & percentages

• Balls where runs scored > 6

• Player-specific stats (e.g., BB McCullum, Rohit Sharma)

• Bowler stats (e.g., most valid balls bowled)

Team-wise extra runs conceded

Unique bowlers for specific teams (e.g., Mumbai Indians)

Technologies Used

• Python

• NumPy

• Pandas

• Jupyter Notebook
• Excel reader libraries (openpyxl, xlrd)

Key Insights from the Analysis

• Total unique batsmen: 516

• Total extra runs in IPL: 12004

• Total runs scored in IPL history: 235,290+

• Most valid deliveries bowled: Harbhajan Singh

• Most balls faced: Virat Kohli
